# Final Validation Oral Defense Script

The methodology was amended at the reporting level, not by erasing the original stored case definition. The frozen March 3 -> March 6 Mindoro base-case YAML is still preserved for provenance, but the promoted thesis-facing primary validation row is now Phase 3B1, the March 13 -> March 14 public-validation pair, because it is the strongest current next-day public-validation row in the package when its shared-imagery caveat is stated honestly.

- Drifters are Phase 1 transport-calibration evidence. They support historical and regional baseline selection over the 2016-2022 validation window rather than serving as spill-truth data for the Mindoro event.
- Observed spill masks are the truth/reference layer for spill-case validation. In Mindoro they define the scored observation geometry, and in DWH they remain the truth masks for the external transfer-validation case.
- Phase 3A is a cross-model comparator. PyGNOME is kept there as a comparator-only branch, never as truth and never as a replacement for the observed-mask validation story.
- March 13 -> March 14 is promoted because it is the clearest current public-validation row for thesis reporting, but it is still a reinitialization-based pair and must carry the caveat that both NOAA products cite March 12 WorldView-3 imagery.
- March 6 is still kept because removing it would blur the evidence trail. It remains the legacy honesty row that shows the sparse strict reference case rather than disappearing just because it is no longer the primary row.
- Phase 1 remains a 2016-2022 historical and regional calibration window. The 2021 cases are useful support or demo context, but they are not the official final Phase 1 evidence base.
- The regional transport-validation box and the Mindoro spill scoring domain answer different questions. One calibrates transport behavior on a broader historical corpus; the other scores the event-specific spill geometry where the observed masks exist.
- DWH remains a separate external transfer-validation case. It strengthens the argument that the workflow transfers to a richer public-observation setting, but it does not replace Mindoro as the main Philippine thesis case.
