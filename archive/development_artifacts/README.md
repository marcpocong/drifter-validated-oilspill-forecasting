# Development Artifact Archive

This folder holds repository-development artifacts that are not scientific evidence.

The files here were retained because they were previously tracked, useful for audit provenance, or both. They are not part of the final paper outputs, not part of the paper-to-output evidence chain, and not a claim of model skill.

Archive lanes:

- `root_debug_logs/`: root-level debug logs moved out of the repository root.
- `temporary_images/`: temporary visual checks and scratch images moved out of the repository root.
- `os_junk/`: tracked operating-system metadata files moved out of source, config, docs, and output trees.

Do not cite these files as scientific artifacts. They exist only to make cleanup traceable without silently deleting tracked clutter.
