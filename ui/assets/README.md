# UI Assets

## Branding Files

Place optional branding assets in this folder:

- `logo.svg`
- `logo.png`
- `logo_icon.svg`
- `logo_icon.png`

The dashboard prefers a transparent background, a wide main logo, and a square optional icon.

Recommended sizing:

- main logo: roughly `1200 x 300` or `1600 x 400`
- icon: roughly `256 x 256` or `512 x 512`

Keep extra whitespace inside the files to a minimum so the logo does not appear surrounded by empty margins.

If no logo is present, the app falls back to clean text-only branding and still loads normally.
