Place optional UI font files here to enable `Ideal Sans` in the Streamlit dashboard.

Preferred filenames:

- `IdealSans-Regular.woff2`
- `IdealSans-Bold.woff2`

Supported fallback formats:

- `IdealSans-Regular.woff`
- `IdealSans-Bold.woff`
- `IdealSans-Regular.ttf`
- `IdealSans-Bold.ttf`
- `IdealSans-Regular.otf`
- `IdealSans-Bold.otf`

If these files are absent, the UI automatically falls back to `Arial`, `Helvetica`, and `sans-serif`.
